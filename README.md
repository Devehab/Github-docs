# دليل تشغيل صفحات HTML على سيرفر GitHub (GitHub Pages)

مرحباً بك! هذا الدليل يشرح لك خطوة بخطوة كيف تقوم بإنشاء صفحات HTML وتجعلها تعمل مباشرة (Live) على الإنترنت باستخدام خدمة **GitHub Pages** المجانية، ثم كيفية رفع ملفاتك إلى **GitHub**.

---

## 🚀 الخطوة 1: إنشاء ملفات الموقع (HTML)
ليعمل موقعك بشكل تلقائي على GitHub Pages، يجب أن يكون الملف الرئيسي للموقع باسم **`index.html`** وفي المجلد الرئيسي (Root) للمشروع.

1. قم بإنشاء ملف `index.html` في مجلد مشروعك.
2. اكتب فيه كود HTML الخاص بك. (لقد قمنا بإنشاء نموذج صفحة بسيطة وأنيقة لك في هذا المجلد).

---

## 🛠️ الخطوة 2: تهيئة Git في مجلد المشروع (لوكال)
إذا لم تكن قد قمت بتهيئة Git بعد في جهازك لهذا المجلد، افتح مبادئ الأوامر (Terminal) في مجلد المشروع ونفذ التالي:

```bash
# 1. تهيئة مستودع Git جديد
git init

# 2. إضافة جميع الملفات للـ Staging area
git add .

# 3. حفظ التغييرات (Commit) لأول مرة
git commit -m "Initial commit: Add index.html and README"

# 4. تعيين الفرع الرئيسي ليكون باسم main
git branch -M main
```

---

## 🌐 الخطوة 3: إنشاء مستودع (Repository) على GitHub وربطه
1. اذهب إلى موقع [GitHub](https://github.com) وقم بتسجيل الدخول.
2. اضغط على زر **New** لإنشاء مستودع جديد.
3. اسمّ المستودع بأي اسم تحبه (مثلاً: `my-live-website`).
4. **تنبيه:** اترك خيارات (Add a README, Add .gitignore) غير محددة لتجنب أي تعارض، لأننا أنشأنا الملفات بالفعل محلياً.
5. بعد إنشاء المستودع، انسخ رابط المستودع (رابط HTTPS أو SSH)، وسيكون بالشكل التالي:
   `https://github.com/YOUR_USERNAME/YOUR_REPO_NAME.git`
6. في الـ Terminal الخاص بك، قم بربط المجلد المحلي بالمستودع على موقع GitHub باستخدام الأمر التالي:

```bash
git remote add origin https://github.com/YOUR_USERNAME/YOUR_REPO_NAME.git
```
*(استبدل الرابط أعلاه برابط مستودعك الفعلي)*

7. الآن، قم برفع الملفات إلى GitHub:

```bash
git push -u origin main
```

---

## ⚡ الخطوة 4: تفعيل GitHub Pages لجعل الموقع Live
بمجرد رفع الملفات إلى GitHub، اتبع الخطوات التالية لتفعيل الرابط المباشر:

1. افتح صفحة المستودع الخاص بك على موقع **GitHub**.
2. اذهب إلى تبويب **Settings** (الإعدادات) في القائمة العلوية.
3. من القائمة الجانبية اليسرى، اختر **Pages**.
4. تحت قسم **Build and deployment** وفي خيار **Source**:
   - تأكد من اختيار **Deploy from a branch**.
5. تحت قسم **Branch**:
   - اختر الفرع **`main`** (بدلاً من None).
   - اترك المجلد الافتراضي **`/ (root)`** كما هو.
   - اضغط على زر **Save**.
6. انتظر حوالي دقيقة أو دقيقتين، ثم قم بتحديث الصفحة. ستظهر لك رسالة في الأعلى تحتوي على رابط موقعك المباشر، مثل:
   `https://YOUR_USERNAME.github.io/YOUR_REPO_NAME/`

مبروك! الآن أي شخص في العالم يستطيع زيارة صفحتك عبر هذا الرابط. 🎉

---

## 🔄 تحديث الموقع مستقبلاً
إذا قمت بأي تعديل على ملفات HTML في جهازك وتريد تحديث الموقع المباشر، ما عليك سوى كتابة الأوامر التالية في الـ Terminal:

```bash
git add .
git commit -m "Update website content"
git push origin main
```
سيقوم GitHub بتحديث الموقع تلقائياً خلال ثوانٍ معدودة بعد الرفع!
